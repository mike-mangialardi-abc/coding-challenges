# Shopping Cart — C#

## Prerequisites

- .NET 8 SDK (`dotnet --version` should report 8.x)

## Install dependencies and run tests

```bash
dotnet test
```

The first invocation will restore NuGet packages and build the project. Subsequent runs are faster.

You should initially see **10 failing tests**. Your goal is to make them all pass by implementing the `ShoppingCart` class in `ShoppingCart.cs`.

## Files

- `ShoppingCart.cs` — the class you need to implement. Method signatures are already there.
- `ShoppingCartTests.cs` — the test suite. Do not modify.
- `ShoppingCart.csproj` — project file. Do not modify.
