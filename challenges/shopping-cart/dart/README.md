# Shopping Cart — Dart

## Prerequisites

- Dart SDK 3.x (`dart --version`)

## Install dependencies and run tests

```bash
dart pub get
dart test
```

You should initially see **10 failing tests**. Your goal is to make them all pass by implementing the `ShoppingCart` class in `lib/shopping_cart.dart`.

## Files

- `lib/shopping_cart.dart` — the class you need to implement.
- `test/shopping_cart_test.dart` — the test suite. Do not modify.
- `pubspec.yaml` — project config. Do not modify.
