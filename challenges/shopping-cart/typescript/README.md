# Shopping Cart — TypeScript

## Prerequisites

- Node.js 20+
- npm (bundled with Node)

## Install dependencies and run tests

```bash
npm install
npm test
```

You should initially see **10 failing tests**. Your goal is to make them all pass by implementing the `ShoppingCart` class in `src/shoppingCart.ts`.

## Files

- `src/shoppingCart.ts` — the class you need to implement.
- `src/shoppingCart.test.ts` — the test suite. Do not modify.
- `package.json`, `tsconfig.json` — project config. Do not modify.
