# Shopping Cart — Ruby

## Prerequisites

- Ruby 3.x (`ruby --version`)
- Bundler (`gem install bundler` if missing)

## Install dependencies and run tests

```bash
bundle install
bundle exec rspec
```

You should initially see **10 failing tests**. Your goal is to make them all pass by implementing the `ShoppingCart` class in `lib/shopping_cart.rb`.

## Files

- `lib/shopping_cart.rb` — the class you need to implement.
- `spec/shopping_cart_spec.rb` — the test suite. Do not modify.
- `Gemfile`, `.rspec` — project config. Do not modify.
