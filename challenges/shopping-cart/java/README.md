# Shopping Cart — Java

## Prerequisites

- JDK 17 or newer (`java -version`)
- Maven 3.9+ (`mvn -v`)

## Install dependencies and run tests

```bash
mvn test
```

The first run will download dependencies. Subsequent runs are fast.

You should initially see **10 failing tests**. Your goal is to make them all pass by implementing `ShoppingCart` in `src/main/java/com/example/shoppingcart/ShoppingCart.java`.

## Files

- `src/main/java/com/example/shoppingcart/ShoppingCart.java` — the class you need to implement.
- `src/test/java/com/example/shoppingcart/ShoppingCartTest.java` — the test suite. Do not modify.
- `pom.xml` — Maven config. Do not modify.
