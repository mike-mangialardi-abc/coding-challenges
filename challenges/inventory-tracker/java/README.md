# Inventory Tracker — Java

## Prerequisites

- JDK 17 or newer
- Maven 3.9+

## Install dependencies and run tests

```bash
mvn test
```

You should initially see **10 failing tests**. Your goal is to make them all pass by implementing `InventoryTracker` in `src/main/java/com/example/inventory/InventoryTracker.java`.

## Files

- `src/main/java/com/example/inventory/InventoryTracker.java` — the class you need to implement.
- `src/test/java/com/example/inventory/InventoryTrackerTest.java` — the test suite. Do not modify.
- `pom.xml` — Maven config. Do not modify.
