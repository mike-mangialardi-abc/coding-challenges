# Inventory Tracker — C#

## Prerequisites

- .NET 8 SDK

## Install dependencies and run tests

```bash
dotnet test
```

You should initially see **10 failing tests**. Your goal is to make them all pass by implementing `InventoryTracker` in `InventoryTracker.cs`.

## Files

- `InventoryTracker.cs` — the class you need to implement.
- `InventoryTrackerTests.cs` — the test suite. Do not modify.
- `InventoryTracker.csproj` — project file. Do not modify.

Read the **problem statement** at `../README.md` first.
