# Inventory Tracker — Dart

## Prerequisites

- Dart SDK 3.x

## Install dependencies and run tests

```bash
dart pub get
dart test
```

You should initially see **10 failing tests**. Your goal is to make them all pass by implementing `InventoryTracker` in `lib/inventory_tracker.dart`.

## Files

- `lib/inventory_tracker.dart` — the class you need to implement.
- `test/inventory_tracker_test.dart` — the test suite. Do not modify.
- `pubspec.yaml` — project config. Do not modify.
