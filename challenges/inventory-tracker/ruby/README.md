# Inventory Tracker — Ruby

## Prerequisites

- Ruby 3.x
- Bundler

## Install dependencies and run tests

```bash
bundle install
bundle exec rspec
```

You should initially see **10 failing tests**. Your goal is to make them all pass by implementing `InventoryTracker` in `lib/inventory_tracker.rb`.

## Files

- `lib/inventory_tracker.rb` — the class you need to implement.
- `spec/inventory_tracker_spec.rb` — the test suite. Do not modify.
- `Gemfile`, `.rspec` — project config. Do not modify.
