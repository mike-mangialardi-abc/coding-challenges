# Rate Limiter — TypeScript

## Prerequisites

- Node.js 20+
- npm

## Install dependencies and run tests

```bash
npm install
npm test
```

You should initially see **10 failing tests**. Your goal is to make them all pass by implementing `RateLimiter` in `src/rateLimiter.ts`.

## Files

- `src/rateLimiter.ts` — the class you need to implement.
- `src/rateLimiter.test.ts` — the test suite. Do not modify.
- `package.json`, `tsconfig.json` — project config. Do not modify.
