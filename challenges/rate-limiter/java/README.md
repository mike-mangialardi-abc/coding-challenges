# Rate Limiter — Java

## Prerequisites

- JDK 17+
- Maven 3.8+

## Install dependencies and run tests

```bash
mvn test
```

You should initially see **10 failing tests**. Your goal is to make them all pass by implementing `RateLimiter` in `src/main/java/com/example/ratelimiter/RateLimiter.java`.

## Files

- `src/main/java/com/example/ratelimiter/RateLimiter.java` — the class you need to implement.
- `src/test/java/com/example/ratelimiter/RateLimiterTest.java` — the test suite. Do not modify.
- `pom.xml` — Maven config. Do not modify.
