# Rate Limiter — Dart

## Prerequisites

- Dart SDK 3.x

## Install dependencies and run tests

```bash
dart pub get
dart test
```

You should initially see **10 failing tests**. Your goal is to make them all pass by implementing `RateLimiter` in `lib/rate_limiter.dart`.

## Files

- `lib/rate_limiter.dart` — the class you need to implement.
- `test/rate_limiter_test.dart` — the test suite. Do not modify.
- `pubspec.yaml` — package config. Do not modify.
