# Rate Limiter — Ruby

## Prerequisites

- Ruby 3.x
- Bundler (`gem install bundler`)

## Install dependencies and run tests

```bash
bundle install
bundle exec rspec
```

You should initially see **10 failing tests**. Your goal is to make them all pass by implementing `RateLimiter` in `lib/rate_limiter.rb`.

## Files

- `lib/rate_limiter.rb` — the class you need to implement.
- `spec/rate_limiter_spec.rb` — the test suite. Do not modify.
- `Gemfile`, `.rspec`, `spec/spec_helper.rb` — project config. Do not modify.
