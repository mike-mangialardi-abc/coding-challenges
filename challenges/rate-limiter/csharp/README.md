# Rate Limiter — C#

## Prerequisites

- .NET 8 SDK

## Install dependencies and run tests

```bash
dotnet test
```

You should initially see **10 failing tests**. Your goal is to make them all pass by implementing `RateLimiter` in `RateLimiter.cs`.

## Files

- `RateLimiter.cs` — the class you need to implement.
- `RateLimiterTests.cs` — the test suite. Do not modify.
- `RateLimiter.csproj` — project file. Do not modify.
